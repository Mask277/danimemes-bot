# danimemes-bot

No build step — static HTML/CSS/JS served directly. Open `index.html` in a browser.

## Architecture

- Two themes toggled by `DarkMode()` in `index.js`: "Arrakis" (default) and "Night City" (Cyberpunk glitch mode)
- Dialogflow chat widget embedded in `index.html`

## Agent routing

| Task | Agent |
|------|-------|
| JavaScript refactoring or new features | `javascript-pro` |
| Visual UI / styling changes | `ui-designer` |
| Diagnosing a bug | `debugger` |
